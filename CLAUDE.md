# AI Coding Cheatsheet

Site de reference/cheatsheet sur Claude Code, en francais, deploy sur GitHub Pages.

## Stack
- Astro + Tailwind CSS 4 + MDX
- Deploy: GitHub Pages via GitHub Actions

## Design Context

### Users
Developpeurs francophones, debutants et experimentes, qui utilisent ou decouvrent Claude Code.

### Aesthetic Direction
- Light mode, fond sable (#faf8f5), accents orange Claude (#e07c3e), texte brun (#4a3728)
- Inspire par awesomeclaude.ai en version claire
- Typo distinctive, pas Inter/Roboto
- Contenu en francais

### Design Principles
1. Scannable — info trouvable en 3 secondes
2. Copy-paste ready — code blocks soignes et copiables
3. Chaleureux — tons sable/terre, pas de gris clinique
4. Multi-pages navigable
5. Progressive — bases pour debutants, details pour experimentes
